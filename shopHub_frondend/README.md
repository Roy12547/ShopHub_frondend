# shopHub_frondend


## Node

```shell
nvm v18.17
```


### Install Packages
```shell
pnpm install
```

### Webpack
1. 開發模式
```shell
pnpm run start
```
本地開發網址: http://localhost:3000/

2. 打包
```shell
pnpm run build
```
